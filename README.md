# 실행방법
``.env.example``를 복사하여 ``.env``로 이름을 바꿔줍니다<br>
``.env``를 설정 후 아래 명령어를 실행시킵니다.
```
php artisan migrate
php artisan serve
```
``php artisan migrate``는 데이터베이스의 테이블을 생성합니다<br>
``php artisan serve``는 라라벨을 실행시킵니다.
